# se-week11-site

実際に服を着用しなくても、ソフトウェア上で今日着る服のコーディネートを考えられるアプリです。服やコーディネートを作成・保存し、一覧表示も可能です。
コーディネートの際の服の配置の精度・画面構成の色どりや視覚的快適さが課題です。

# テスト結果報告書
## 現在動く機能
- [x] 服の画像を複数枚保存できる
- [x] 登録した服の一覧表示
- [x] 登録済みの服を複数枚合わせ、用途に合わせたスタイリング作成・保存が可能
- [x] 保存したコーディネートの一覧表示
- [x] 登録した服・コーディネートの削除
- [ ] 登録済みの服・コーディネートの編集機能は未実装
## 起動方法
python app.py
## 動作確認
1. ブラウザで http://localhost:5000 を開く
2. 「服を登録」ボタンで対象の服をアップロード
3.「クローゼットを見る」で登録済みの服を一覧表示(カテゴリで絞みこみ可)
4. 「コーディネート作成」で登録済みの服を複数枚選択・組み合わせて保存する
5. 「作成したコーディネートを見る」で保存済みのコーディネートを一覧表示
